# 🍬 Nassau Candy Factory Optimization System

An AI-powered supply chain optimization and decision support system that analyzes historical shipping and sales data to recommend the most efficient factory assignments for products. The system uses machine learning to predict shipping lead times, simulate factory reallocation scenarios, evaluate operational impact, and generate data-driven recommendations to improve logistics performance while maintaining profitability.

## Key Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Lead Time Prediction using Machine Learning
- Route Clustering Analysis
- Factory Reallocation Simulation
- What-If Scenario Analysis
- Recommendation Engine
- Risk & Impact Assessment
- Interactive Streamlit Dashboard

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Plotly
- Streamlit
- Joblib

## Business Impact
The system helps organizations identify optimal product-to-factory assignments, reduce shipping lead times, improve supply chain efficiency, and support data-driven operational decisions through predictive analytics and simulation.
